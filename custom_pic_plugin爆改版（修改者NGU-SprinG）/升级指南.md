# Custom Pic Plugin - 升级指南

本指南适用于已安装原版插件的用户，通过替换文件的方式升级到修改版。

## 📦 需要替换的文件

以下文件需要用修改版的文件替换原版文件：

### 核心文件
| 文件路径 | 说明 |
|---------|------|
| `plugin.py` | 主插件文件，版本号更新至3.3.5，新增依赖包配置 |
| `core/pic_action.py` | 新增智能参考搜索功能、自拍负面提示词处理 |
| `core/pic_command.py` | 新增 `/dr auto_selfie` 命令 |
| `core/runtime_state.py` | 新增定时自拍相关状态管理 |

### 配置文件
| 文件路径 | 说明 |
|---------|------|
| `_manifest.json` | 版本号更新至3.3.5 |

## 🔧 升级步骤

### 方法一：手动替换（推荐）

1. **备份原版插件**
   ```bash
   cd MaiBot/plugins
   cp -r custom_pic_plugin custom_pic_plugin.backup
   ```

2. **替换核心文件**
   - 将修改版的以下文件复制到原版插件目录覆盖原文件：
     - `plugin.py`
     - `core/pic_action.py`
     - `core/pic_command.py`
     - `core/runtime_state.py`
     - `_manifest.json`

3. **新增功能文件**
   - 将修改版的以下文件复制到原版插件目录：
     - `core/auto_selfie_manager.py`
     - `core/image_search_adapter.py`
     - `core/vision_analyzer.py`
     - `core/search_engines/` 整个目录
     - `功能添加说明.md`
     - `升级指南.md`

4. **重启MaiBot**
   - 重启后MaiBot会自动生成包含新功能的配置项

### 方法二：直接克隆（简单）

```bash
cd MaiBot/plugins
# 备份原版
mv custom_pic_plugin custom_pic_plugin.backup
# 克隆修改版
git clone https://github.com/nguspring/custom_pic_plugin
```

## ⚙️ 配置更新

升级后，配置文件会自动新增以下配置节：

### 定时自拍配置
```toml
[auto_selfie]
enabled = false
interval_minutes = 30
ask_message = "这张照片怎么样？"
selfie_style = "standard"
model_id = "model1"
```

### 智能参考搜索配置
```toml
[search_reference]
enabled = false
vision_api_key = ""
vision_base_url = "https://api.openai.com/v1"
vision_model = "gpt-4o"
```

### 自拍负面提示词配置
```toml
[selfie]
negative_prompt = ""
```

## 📦 依赖安装

修改版新增了以下Python依赖包，需要手动安装：

```bash
pip install aiohttp>=3.8.0
pip install beautifulsoup4>=4.11.0
pip install lxml>=4.9.0
pip install ddgs
```

## ⚠️ 注意事项

1. **配置文件保留**：升级时不会覆盖你的 `config.toml`，新配置项会自动添加
2. **备份重要**：升级前务必备份原版插件
3. **依赖安装**：确保安装了新增的依赖包
4. **版本兼容**：修改版基于原版v3.3.3开发，建议从v3.3.3或更高版本升级

## 🔄 回滚方法

如果升级后遇到问题，可以快速回滚：

```bash
cd MaiBot/plugins
# 停止MaiBot
rm -rf custom_pic_plugin
mv custom_pic_plugin.backup custom_pic_plugin
# 重启MaiBot
```
