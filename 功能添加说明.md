## 新增功能详解

### 1. 📸 定时自拍功能

#### 功能描述
改版插件新增了定时自拍功能，让MaiBot可以定时自动发送自拍照片，使Bot更像真人。发完自拍后会自动询问用户照片怎么样。

#### 新增文件
- [`core/auto_selfie_manager.py`](core/auto_selfie_manager.py) - 定时自拍任务管理器

#### 配置项
```toml
[auto_selfie]
enabled = false  # 是否启用定时自拍功能
interval_minutes = 30  # 定时自拍间隔时间（分钟），建议10-120分钟
ask_message = "这张照片怎么样？"  # 发完自拍后自动发送的询问语
selfie_style = "standard"  # 自拍风格：standard=标准自拍，mirror=对镜自拍
model_id = "model1"  # 定时自拍使用的模型ID
```

#### 新增命令
```
/dr auto_selfie on [间隔分钟]  # 启动定时自拍（默认30分钟）
/dr auto_selfie off             # 停止定时自拍
/dr auto_selfie status          # 查看定时自拍状态
/dr auto_selfie config          # 查看定时自拍配置
```

#### 功能特性
- 按聊天流分别管理定时自拍任务
- 支持自定义定时间隔（5-300分钟）
- 支持两种自拍风格：标准自拍（前置摄像头）和对镜自拍
- 发完自拍后自动发送询问语（可自定义或随机选择模板）
- 所有状态仅在内存中保持，重启后重置

### 2. 🔍 智能参考搜索功能

#### 功能描述
当用户提到冷门角色名时，插件会自动联网搜索该角色的参考图，并使用视觉AI提取特征（发色、服装等）合并到提示词中，解决模型不认识角色的问题。

#### 新增文件
- [`core/image_search_adapter.py`](core/image_search_adapter.py) - 图片搜索适配器
- [`core/vision_analyzer.py`](core/vision_analyzer.py) - 视觉分析客户端

#### 配置项
```toml
[search_reference]
enabled = false  # 是否启用智能参考搜索功能
vision_api_key = ""  # 用于识图的API Key（支持视觉的模型，如 gpt-4o）
vision_base_url = "https://api.openai.com/v1"  # 识图API地址
vision_model = "gpt-4o"  # 视觉模型名称
```

#### 功能特性
- 动态导入联网搜索插件的搜索引擎（Bing、搜狗、DuckDuckGo）
- 自动搜索角色图片并提取视觉特征
- 将提取的特征以权重1.3合并到用户提示词中
- 支持多种搜索引擎，按优先级尝试
- 搜索失败不影响正常绘图流程

#### 工作流程
1. 用户输入角色名称（如"画一个XXX角色"）
2. 插件检测到智能参考搜索已启用
3. 调用图片搜索适配器搜索角色参考图
4. 使用视觉AI分析图片，提取特征（如：red hair, white hat...）
5. 将特征合并到提示词：`原描述, (提取的特征:1.3)`
6. 继续正常的绘图流程

### 3. 🎨 自拍负面提示词配置

#### 功能描述
改版插件新增了自拍模式专用的负面提示词配置，可以单独设置自拍时的负面提示词，会叠加在模型默认负面提示词上。

#### 配置项
```toml
[selfie]
enabled = true  # 是否启用自拍模式功能
reference_image_path = ""  # 自拍参考图片路径
prompt_prefix = ""  # 自拍模式专用提示词前缀
negative_prompt = ""  # 自拍模式专用的负面提示词（新增）
```

#### 功能特性
- 自拍专用负面提示词会与模型默认负面提示词合并
- 支持在自拍模式下单独控制负面提示词
- 适用于需要特定负面提示词的自拍场景

### 4. 🔧 内置搜索引擎功能（v3.3.5新增）

#### 功能描述
v3.3.5版本将联网搜索插件的搜图功能内置到AI绘图插件中，无需依赖外部联网搜索插件即可使用智能参考搜索功能。

#### 新增文件
- [`core/search_engines/__init__.py`](core/search_engines/__init__.py) - 搜索引擎模块初始化
- [`core/search_engines/base.py`](core/search_engines/base.py) - 搜索引擎基类
- [`core/search_engines/bing.py`](core/search_engines/bing.py) - Bing搜索引擎实现
- [`core/search_engines/sogou.py`](core/search_engines/sogou.py) - 搜狗搜索引擎实现
- [`core/search_engines/duckduckgo.py`](core/search_engines/duckduckgo.py) - DuckDuckGo搜索引擎实现

#### 修改文件
- [`core/image_search_adapter.py`](core/image_search_adapter.py) - 更新为使用内置搜索引擎
- [`_manifest.json`](_manifest.json) - 版本号更新至3.3.5

#### 功能特性
- **完全独立**：不再依赖外部联网搜索插件
- **多引擎支持**：内置Bing、搜狗、DuckDuckGo三种搜索引擎
- **智能回退**：按优先级尝试不同搜索引擎，提高成功率
- **国内可用**：Bing和搜狗无需科学上网即可使用

#### 依赖要求
内置搜索引擎需要以下Python依赖包：
```
aiohttp>=3.8.0
beautifulsoup4>=4.11.0
lxml>=4.9.0
ddgs
```

#### 工作流程
1. 用户输入角色名称（如"画一个XXX角色"）
2. 插件检测到智能参考搜索已启用
3. 按优先级尝试内置搜索引擎（Bing → 搜狗 → DuckDuckGo）
4. 使用视觉AI分析图片，提取特征
5. 将特征合并到提示词：`原描述, (提取的特征:1.3)`
6. 继续正常的绘图流程

## 注意事项

1. **定时自拍功能**
   - 定时间隔建议设置为10-120分钟，太频繁可能会打扰用户
   - 所有状态仅在内存中保持，重启MaiBot后需要重新启动定时自拍
   - 需要先在配置文件中启用 `auto_selfie.enabled`

2. **智能参考搜索功能**
   - **v3.3.5内置版**：无需联网搜索插件，已内置搜索引擎
   - **v3.3.4外部插件版**：需要联网搜索插件支持
   - 需要配置视觉API的Key和地址
   - 搜索失败不会影响正常绘图流程
   - 仅在非自拍模式下生效
   - 内置搜索引擎需要安装额外依赖包（见依赖说明）

3. **自拍负面提示词**
   - 会与模型默认负面提示词合并
   - 格式：`原有负面词, 自拍专用负面词`
